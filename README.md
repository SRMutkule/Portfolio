# My Personal Portfolio

A **modern and responsive personal portfolio website** built with **HTML, CSS, and JavaScript**. It showcases my **skills, projects, and contact information** with interactive features and dynamic content.

---

## 🚀 Features

- **Typing Animation** – Dynamic role display in the hero section  
- **Scroll Animations** – Fade-in and reveal effects for sections and cards  
- **Responsive Navbar** – Active link highlighting and mobile-friendly menu  
- **GitHub API Integration** – Displays latest projects dynamically  
- **Skill Bars** – Interactive skill level indicators that animate on scroll  
- **Contact Form** – Validates user input with success/error feedback messages  
- **Modern UI Design** – Glassmorphism, gradient texts, and responsive layouts  

---

## 🛠️ Technologies Used

- **HTML5 & CSS3** – Semantic markup and modern styling techniques  
- **JavaScript (ES6+)** – DOM manipulation, animations, and API integration  
- **GitHub API** – Fetches and displays repositories dynamically  
- **Responsive Design** – Mobile-first layout and media queries
